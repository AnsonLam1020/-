# 遊戲中心 PRD (產品需求文檔)

## 基本資訊

- 網站名稱：Anson's Game Center
- 設計風格：深色科技感 + 明亮可愛風 (雙模式切換)

## 功能需求

### 核心功能
1. **遊戲分類與篩選**
   - 按遊戲類型分類瀏覽
   - 搜尋功能 (即時搜尋)
   - 標籤過濾

2. **主題切換 (黑暗/明亮模式)**
   - 一鍵切換按鈕
   - 自動偵測系統偏好
   - 保存用戶選擇 (localStorage)

3. **響應式設計**
   - 支援桌面、平板、手機
   - 適應不同螢幕尺寸
   - 觸控友善介面

4. **遊戲啟動**
   - 無縫跳轉到遊戲頁面
   - 遊戲進度保存 (如適用)
   - 高分記錄 (localStorage)

### 技術規格
- **前端技術**：HTML5, CSS3, JavaScript (ES6+)
- **遊戲技術**：Canvas API, Web Audio API, localStorage
- **相容性**：Chrome, Firefox, Safari, Edge (最新版本)
- **性能要求**：頁面加載時間 < 3秒，遊戲幀率 ≥ 60fps
- **無障礙性**：符合 WCAG 2.1 AA 標準

## 設計規範

### 黑暗模式 (預設)
- 背景：linear-gradient(135deg, #1a1a2e 0%, #16213e 100%)
- 文字顏色：白色 #fff，灰色 #a9b7c6
- 主色調：青色 #4ecdc4，藍色 #45b7d1，粉色 #ff6b6b
- 卡片背景：rgba(45, 45, 68, 0.8)
- 陰影效果：0 10px 20px rgba(0, 0, 0, 0.2)

### 明亮模式
- 背景：linear-gradient(135deg, #f0f8ff 0%, #e6f7ff 100%)
- 文字顏色：深灰 #333，中灰 #666
- 主色調：橙色 #ffa500，綠色 #4CAF50，藍色 #2196F3
- 卡片背景：rgba(255, 255, 255, 0.9)
- 陰影效果：0 5px 15px rgba(0, 0, 0, 0.1)

### 字體規範
- 主要字體：'Segoe UI', 'Microsoft JhengHei', sans-serif
- 遊戲字體：'Microsoft JhengHei', 'Segoe UI', sans-serif
- 字體大小：
  - 標題：2.5-3.5rem (桌面)，2-2.5rem (手機)
  - 正文：1rem
  - 小字：0.85rem
